## 摘要
我们提出无键注意力（Keyless Attention），一种完全消除键投影的注意力机制，仅操作查询和值。这产生了仅值缓存（Value-Only Cache），与标准注意力相比，KV缓存内存和访问开销精确减少50%，同时匹配或超越标准注意力的解码吞吐量。除了效率之外，我们引入了深度m注意力分解（Depth-m Attention Factorization）：标准注意力计算注意力双线性形式的深度-2分解，而无键注意力实现了该家族的深度-m实例。在m=3时，无键注意力通过一个值空间路由矩阵匹配标准注意力的投影矩阵数量，该矩阵替代了键投影，并引入了路由与检索之间的耦合。在五个模型和四种架构（GPT-2 280M、GPT-2 557M、Pythia 410M、Qwen2 1.5B和Llama 3.2 1B）上的实验表明，无键注意力在5个模型中的4个上，在困惑度方面匹配或优于标准QKV注意力。在下游零样本评估（GPT-2 557M）中，无键注意力在5个常识推理基准中的4个上表现更优，同时实现了50%的KV缓存减少。

## Abstract
We propose Keyless Attention, an attention mechanism that eliminates the key projection entirely, operating over queries and values only. This yields a Value-Only Cache that reduces KV cache memory and access overhead by exactly 50% over standard attention, while matching or exceeding standard attention's decode throughput. Beyond efficiency, we introduce Depth-$m$ Attention Factorization: standard attention computes a depth-2 factorization of the attention bilinear form, while Keyless Attention realizes a depth-$m$ instance of this family. At m=3, Keyless Attention matches the projection matrix count of standard attention via a value-space routing matrix that replaces the key projection and introduces a coupling between routing and retrieval. Experiments across five models and four architectures (GPT-2 280M, GPT-2 557M, Pythia 410M, Qwen2 1.5B, and Llama 3.2 1B) show that Keyless Attention matches or outperforms standard QKV attention on perplexity in 4 out of 5 models. On downstream zero-shot evaluation (GPT-2 557M), Keyless Attention outperforms on 4 out of 5 commonsense reasoning benchmarks, while achieving 50% KV cache reduction throughout.

### 1. 核心问题与整体含义（研究动机和背景）
- **核心问题**：Transformer 在自回归推理时，KV 缓存（Key-Value cache）的内存占用和访问开销随序列长度和层数线性增长，成为长上下文推理的主要瓶颈。现有优化方法（如量化、剪枝、系统级管理）都在“事后”压缩缓存，而未质疑其必要性。
- **研究动机**：标准 QKV 注意力同时存储键和值的独立表示，可能存在冗余。受人类认知启发——人类从记忆中检索时只使用单一查询对应单一存储，而非两份独立副本——作者尝试彻底移除键投影，仅依赖查询和值进行注意力计算。
- **整体含义**：提出 Keyless Attention，通过“值空间路由”和“仅值缓存”从源头减少 KV 缓存一半，同时保持或提升模型性能，为高效 Transformer 推理提供新范式。

### 2. 方法论
#### 2.1 核心思想
- 移除标准注意力中的键投影（`K = XW^K`），直接计算查询（`Q`）和值（`V`）之间的注意力分数：  
  `Attention(Q,V) = softmax(QV^T / sqrt(d_k)) V`。
- 值空间路由：用值投影矩阵代替键投影，使路由信号直接与检索内容耦合，形成归纳偏置——模型倾向于关注其值内容与查询直接相关的 token。

#### 2.2 关键技术细节
- **理论等价性**：定理证明在单头（全秩）和_多头（列满秩 + 子空间条件）下，标准注意力的 Ω = W^Q (W^K)^T 总能被 Keyless 的 ˜W^Q (W^V)^T 精确匹配，因此键投影可被完全吸收，理论表达能力不变。
- **Depth-m 注意力分解**：将标准 QKV 视为深度-2 分解（m=2），Keyless Attention 推广为深度-m 家族。本文主要采用 **QVV(3)**（m=3）：`W^Q = W^{Q1} W^{Q2}`, `W^V` 独立，投影矩阵总数与标准 QKV 相同，推理时可融合 `W^{Q1} W^{Q2}` 为单矩阵，消除额外开销。
- **值空间耦合与梯度纠缠**：由于路由矩阵 `W^{Q2}` 和值矩阵 `W^V` 的梯度相互依赖（均含对方因子），形成隐式正则化，减缓过拟合。
- **仅值缓存**：推理时只缓存每个头或每组的值张量，KV 缓存内存和访问开销精确减少 50%。支持与 MQA、GQA、MLA 等现有方法结合，进一步压缩。
