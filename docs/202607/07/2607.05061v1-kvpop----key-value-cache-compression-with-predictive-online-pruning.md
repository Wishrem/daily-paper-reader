## 摘要
键值缓存增长是自回归解码的主要瓶颈，因为内存和带宽随上下文长度线性扩展。现有的KV驱逐方法通常依赖静态启发式或代理分数，这难以跟踪未来令牌的效用，并且随着相关性的变化导致脆弱的驱逐。为了解决这个问题，我们引入了KVpop，它通过直接监督保留或丢弃决策来学习固定预算的KV驱逐策略。该评分器针对一种新颖的未来注意力目标进行训练，该目标在不生成密集注意力图的情况下高效计算。我们进一步引入了基于延迟记忆的评分器，这在学习型驱逐方法中独一无二，它将评分延迟固定步数以利用近期未来上下文。在AIME和HMMT数学推理任务上，KVpop在Qwen3-4B模型上，75%的KV缓存压缩下保留了98%的全注意力性能，88%压缩下保留了97%，持续优于已有的驱逐基线。Qwen3-8B显示出更强结果，接近全教师模型性能。这些结果表明，使用未来注意力信号监督驱逐可以在保持质量的同时降低内存成本。

## Abstract
Key-value (KV) cache growth is a major bottleneck in autoregressive decoding, as memory and bandwidth scale linearly with context length. Existing KV eviction methods often rely on static heuristics or proxy scores, which poorly track future token utility and cause brittle eviction as relevance shifts. To address this, we introduce KVpop, which learns a fixed-budget KV eviction policy by directly supervising the keep-or-drop decision. The scorer is trained against a novel future-attention target, computed efficiently without materializing dense attention maps. We further introduce a delayed memory-based scorer that, uniquely among learned eviction methods, defers scoring for a fixed number of steps to exploit near-future context. On AIME and HMMT mathematical reasoning, KVpop retains 98% of full-attention performance on Qwen3-4B at 75% KV cache compression and 97% at 88% compression, consistently outperforming established eviction baselines. Qwen3-8B shows even stronger results, reaching near-full teacher performance. These results show that supervising eviction with future-attention signals cuts memory costs while maintaining quality.

- **核心问题**：Transformer 自回归解码中，KV 缓存（Key-Value Cache）随上下文长度线性增长，成为长序列推理的主要内存和带宽瓶颈。现有 KV 驱逐方法（如 StreamingLLM、TOVA）依赖静态启发式或局部注意力分数，无法准确预测令牌的未来效用，导致缓存内容选择脆弱、模型性能下降。
- **整体含义**：论文旨在通过学习一个固定预算的、监督式的驱逐策略，在显著降低 KV 缓存内存占用（最高 88% 压缩率）的同时，保持甚至接近全注意力模型的推理质量，为长上下文 LLM 的高效部署提供新范式。

### 2. 方法论：核心思想、关键技术细节

- **核心思想**：KVpop 是一种稀疏注意力改造方法，为每个 KV 头分配固定预算（s 个 sink 令牌 + w 个保护窗口令牌 + k 个长期 top-k 令牌）。通过轻量级评分器在线预测每个令牌的重要性，并利用未来注意力目标监督这些评分，使得驱逐决策直接优化未来效用。
- **关键技术细节**：
  - **未来注意力目标**：对于每个令牌，定义其离开保护窗口后接收到的平均未来注意力质量作为训练目标。通过转置注意力（transposed attention）高效计算，避免生成密集注意力图。
  - **边界感知损失（Boundary-aware loss）**：在每次查询位置，只关注新离开保护窗口的令牌与教师模型中保留的边界令牌之间的分数对比，使用排序损失（pairwise logistic loss）训练。
  - **延迟状态评分器**：基于 mLSTM 的记忆评分器，允许在令牌离开保护窗口前积累一段时间内的上下文证据，再做出保留/丢弃决策，从而利用近未来信息。
  - **高效实现**：使用 Fenwick 树实现在线运行 top-k 阈值计算，与 FlexAttention 结合，无需存储完整掩码矩阵。
- **流程概要**：训练时，每个 KV 头通过评分器输出重要性分数，结合衰减（recency bias）构造有效分数，然后通过 Fenwick 树获得每查询的 top-k 掩码。稀疏注意力计算后，再用转置注意力获取未来注意力目标，计算边界损失。学生模型通过 KL 散度蒸馏教师模型并联合优化评分器。

### 7. 优点

- **方法创新**：首次将未来注意力质量作为直接监督信号，并设计了高效转置注意力实现；延迟状态评分器利用保护窗口期间积累的证据，这是已有方法未做到的。
- **高效实现**：Fenwick tree + FlexAttention 避免了 O(S²) 掩码，训练额外开销小，推理无额外成本。
- **强实验结果**：在多种压缩率和不同能力基座上 consistently 超越所有对比方法，尤其是在高压缩率下仍保持极高性能。
- **可解释性**：提供可视化分析，帮助理解模型学到的驱逐模式，增加可信度。
- **公平对比**：严格控制 DMS 参数数量匹配，隔离训练策略差异的影响；同时训练 StreamingLLM+ 消除仅蒸馏带来的混淆。

### 8. 不足与局限

- **架构局限性**：KVpop 仅作为后训练改造（retrofit）应用于预训练 Transformer，而非从头训练的压缩缓存架构，可能无法完全挖掘架构协同优化潜力。
- **评分器探索有限**：仅深入探索了 mLSTM 作为状态评分器，其他记忆模型（如 Mamba、DeltaNet）未评估。
- **预算均匀性**：所有 KV 头采用相同固定预算，而模型不同层/头对长程依赖需求不均，可能导致某些头浪费预算、另一些头不足。作者在局限中承认这一点，并提及未来可探索混合密集-稀疏层。
- **实验覆盖**：只在 Qwen3 系列（4B 和 8B）上验证，未在更大规模模型（如 70B）或不同架构（如 LLaMA）上测试。训练数据仅来自数学推理，泛化到代码和科学问答但未包括更开放的长文本场景（如文档摘要、多轮对话）。
- **基线范围**：未与稀疏检索方法（如 Quest、TokenButler）直接对比，虽然论文解释这些方法不强制固定内存，但实际工业部署中内存仍有限制，对比不够充分。
